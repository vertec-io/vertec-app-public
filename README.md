# Vertec

Welcome to Vertec, a software technology company focused on industrial automation in the Industrial Internet of Things (IIoT) space.

## Overview

Vertec aims to enable the creation and interconnection of "vertices" in a fully distributed IIoT network. Our platform leverages the power of converging technologies to revolutionize the industrial automation industry. With Vertec, you can unleash your creativity, build innovative solutions, and shape the future of connected industries.

## Folder Structure

The repository is organized using the following folder structure:

- `backend/`: Contains the backend services implemented in Rust.
- `frontend/`: Contains the frontend application developed using Next.js and React.

For more details about the folder structure, refer to the [detailed folder structure](./folder-structure.md) documentation.

## Getting Started

To get started with the Vertec project, follow these steps:

1. Clone the repository: `git clone https://github.com/your-username/Vertec.git`
2. Set up the backend services by following the instructions in the respective microservice directories.
3. Set up the frontend application by following the instructions in the respective microservice directories.
4. Start the backend services and frontend application according to their individual setup instructions.

For detailed instructions on setting up and running the backend services and frontend application, refer to the respective documentation in the microservice directories.
